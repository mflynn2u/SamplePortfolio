Machine Learning - EMNIST dataset


Machine learning model using the emnist dataset from Kaggle.

Using a Convolutional Neural Network and a Random Forest model (for comparison) a hand-drawn image from a web application is identified as a letter or number out of 47 different options (classes).

The hand drawn image is sent to a Flask application where it is passed to tue neural network model for identification.
The image is also processed within its boundaries and then passed to the model for identification.
