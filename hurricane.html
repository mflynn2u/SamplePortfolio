Pick-a-hurricane

Select a Hurricane from the Timeline, once selected information about the event will be displayed.

Timeline includes Hurricanes and Typhoons from 1851 to 2014

Windspeed and path is displayed for each event selected.


This project was made with : Python (Flask, SQLAlchemy )
HTML, CSS, JavaScript ( Leaflet, Timeline, Plotly, Moment )

Data : Generated GeoJson, Json, csv (Kaggle), SqlLite

Tools: Git.
