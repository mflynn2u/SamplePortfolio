Correlations of over 500 random cities in relation to the Ecuator


Data for this analysis was gathered performing API requests on a single date and time for random cities. Factors gathered where Max. Temperature, Cloudiness, Humidity Wind speed and Wind Direction, these factors, where used in the search for correlations with the city's latitude, keeping in mind that a latitude of the Ecuator is 0.

This project was made with : Python (Pandas, matplotlib, requests, seaborn)
Data : JSON (API)
Tools: Jupyter notebook, Git.
